# Schoolter — London Schools Explorer

A web app to view, search, and analyze all London schools across every borough.

## Features

- **Search** — Find schools by name, borough, or postcode
- **Filter** — Filter by borough, phase, gender, Ofsted rating, faith, funding type, and sixth form availability
- **Sort** — Sort by name, pupil count, or borough
- **Detail View** — Click any school card to see full details
- **Analytics** — Visual charts showing distribution by borough, phase, Ofsted rating, gender, and funding type
- **Compare** — Side-by-side comparison of any two schools

## Data

The dataset includes 140 schools across all 32 London boroughs, covering primary, secondary, all-through, and special schools.

## Getting Started

```bash
cd schoolter
npx serve .
```

Then open `http://localhost:3000` in your browser.

Alternatively, just open `index.html` directly in a browser — no build step required.

## Tech Stack

- Vanilla HTML, CSS, JavaScript
- No dependencies or build tools required
- Responsive design for mobile and desktop
